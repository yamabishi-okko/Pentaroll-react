# 🌙遊び方要約
>通常の五目並べと同じで5個並べた方の勝利（縦・横・斜め）
盤面の端からボールを押しこんで入れてずらしていく
すでにボールがある箇所にも押し込んで入れることができる（6個あるときは押しこみ不可）
