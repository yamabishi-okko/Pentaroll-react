# 🌙遊び方要約
>通常の五目並べと同じで5個並べた方の勝利（縦・横・斜め）<br>
盤面の端からボールを押しこんで入れてずらしていく<br>
すでにボールがある箇所にも押し込んで入れることができる（6個あるときは押しこみ不可）

# 🧸使用技術
>フロントエンドフレームワーク<br>
・React + Vite<br>
フロントエンド言語<br>
・TypeScript<br>
バックエンド関連<br>
・CPU対戦モード実装予定（Python使用）
